# Modern Retro Portfolio

Portfolio modern dengan Retro UI menggunakan Next.js, TypeScript, Tailwind CSS, font Syne + Poppins, dan AOS Blur animation.

## Jalankan Lokal

```bash
npm install
npm run dev
```

## Deploy Vercel

Upload repository ke GitHub, lalu import ke Vercel. Framework akan otomatis terdeteksi sebagai Next.js.

## Edit Konten

Semua konten utama ada di folder:

```bash
lib/settings/*
```

File penting:

- `site.ts`
- `hero.ts`
- `about.ts`
- `skills.ts`
- `education.ts`
- `projects.ts`
- `contact.ts`
